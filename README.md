# Microsoft-Natural-Ergonomic-Keyboard-armrests-rebuild
Simple templates for Microsoft Natural Ergonomic Keyboard armrests faux leather rebuild

The reason I started this project is that I love the Microsoft Natural Ergonomic Keyboard 4000. Unfortunately, over time, the wrist rests on this keyboard wear out, and sadly, this model was discontinued in 2020, so itâ€™s no longer possible to buy a new one.

PDF files contain cutting lines which can be used by plotter, Zund cutting machine etc.
Expects 12x12 pleather tile

Faux leather:
 - selfadhesive
 - Material PU (pleather)
 - thick 0.8 - 0.9 mm (I'm not sure because the original material is in bad shape)

The project is inspired by this video:
https://www.youtube.com/watch?v=Xm1ci6h0W3E&t=213s

Howto:
1. Unscrew lower 10 screws
2. Carefully remove armersts. Slowly pull it up only. Not to sides because holders may broke easily.
3. Peel broken pleather
4. Cut new pleather pieces according to cuttings in PDF folder
5. Stick it well onto the armrests
6. Screw everything back
